# 費率計算方式對比分析

## 🎯 核心問題

**系統費應該：**
- 方案 A：格外收取（投資金額 × 系統費 + 投資金額 × 其他費用）
- 方案 B：包含在內（所有費用加起來 = 投資金額）

---

## 🔧 實際實作方式

### 方案 B 的兩種計算方式

#### 方式 1：租戶設定百分比是相對於剩餘金額（複雜）❌

```typescript
投資金額: 1000 USDT

// 1. 先扣系統費
systemFee = 1000 × 1% = 10 USDT
remaining = 1000 - 10 = 990 USDT

// 2. 租戶的費率是相對於剩餘金額
platformFee = 990 × 2% = 19.8 USDT
agentCommission = 990 × 5% = 49.5 USDT

// 問題：租戶看到的數字不直觀
// 設定 2% 但實際收到 19.8 USDT？
```

#### 方式 2：所有費率都相對於原始投資金額（簡單）⭐

```typescript
投資金額: 1000 USDT

// 所有費率都基於 1000 USDT
systemFee = 1000 × 1% = 10 USDT
platformFee = 1000 × 2% = 20 USDT
agentCommission = 1000 × 5% = 50 USDT

netInvestment = 1000 - 10 - 20 - 50 = 920 USDT

// ✅ 租戶設定 2%，真的收到 20 USDT
// ✅ 代理設定 5%，真的收到 50 USDT
// ✅ 數字直觀
```

## 📝 資料庫設計

### Tenant Entity

```typescript
{
  // 系統費率（由 Platform 設定，租戶不可改）
  systemFeeRate: 1.0,        // %
  
  // 租戶可設定的費率
  cryptoConfig: {
    platformFeeRate: 2.0,    // %
    agentBaseRate: 5.0,      // %
  },
  
  // 計算規則：所有費率都基於原始投資金額
  // totalFeeRate = systemFeeRate + platformFeeRate + agentBaseRate
}
```

### 計算邏輯

```typescript
function calculateFees(investmentAmount: number, tenant: Tenant) {
  const fees = {
    // 所有費率都基於原始投資金額
    systemFee: investmentAmount * (tenant.systemFeeRate / 100),
    platformFee: investmentAmount * (tenant.cryptoConfig.platformFeeRate / 100),
    agentCommission: investmentAmount * (tenant.cryptoConfig.agentBaseRate / 100),
  };
  
  fees.totalFees = fees.systemFee + fees.platformFee + fees.agentCommission;
  fees.netInvestment = investmentAmount - fees.totalFees;
  
  // 驗證：總費率不應超過合理範圍
  const totalFeeRate = tenant.systemFeeRate + 
                       tenant.cryptoConfig.platformFeeRate + 
                       tenant.cryptoConfig.agentBaseRate;
  
  if (totalFeeRate > 20) {
    throw new Error('總費率不能超過 20%');
  }
  
  return fees;
}

// 使用範例
const fees = calculateFees(1000, tenant);
console.log(fees);
/*
{
  systemFee: 10,
  platformFee: 20,
  agentCommission: 50,
  totalFees: 80,
  netInvestment: 920
}
*/
```

## 🎯 總結

**強烈推薦：方案 B（系統費包含在內）**

**核心原則**：
```
投資金額 × 費率 = 實際費用
所有費率基於原始投資金額
總費用 = 系統費 + 平台費 + 代理佣金
淨投資 = 投資金額 - 總費用
```

**關鍵優勢**：
- 完全透明
- 易於理解
- 無爭議
