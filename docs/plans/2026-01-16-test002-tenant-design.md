# test002 租戶頁面設計文檔

## 概述

為 customer-web 應用創建 test002 租戶的完整投資流程頁面，採用專業理財風格，提供固定收益產品投資服務。

## 目標

- 創建完整的投資授權流程（錢包連接 → USDT 授權 → 用戶註冊 → 成功頁面）
- 展示多租戶系統支援不同品牌風格的能力
- 提供測試環境驗證端對端流程

## 設計定位

**品牌風格：** 專業理財型
- 目標用戶：偏好穩健投資的保守型投資者
- 語言風格：正式專業但不生硬，使用「您」稱呼
- 視覺風格：專業商務，強調信任和安全

**產品定位：** 固定收益產品
- 預期年化收益：8-12%
- 投資門檻：100 USDT 起
- 鎖定期限：靈活贖回
- 風險等級：穩健型

## 視覺設計

### 配色方案

**主色調：**
- 深藍色（#1e3a8a, blue-800）- 專業、信任、穩定
- 金色（#f59e0b, amber-500）- 價值、優質、回報
- 深灰色（#1f2937, gray-800）- 文字主色

**輔助色：**
- 淺藍背景（#eff6ff to #dbeafe, blue-50 to blue-100）- 頁面漸層背景
- 淺金背景（#fef3c7, amber-50）- 重點資訊區塊
- 綠色（#059669, emerald-600）- 成功狀態

### 視覺特點

1. **卡片設計**：大圓角（rounded-xl）+ 柔和陰影（shadow-xl）
2. **層次分明**：使用邊框和背景色區分資訊區塊
3. **留白充足**：padding-8 以上，避免擁擠感
4. **專業簡潔**：避免過多裝飾，以內容為主

## 技術架構

### 文件結構

```
apps/customer-web/tenants/
├── test002/
│   ├── index.ts          # 導出 InvestPage 和 SuccessPage
│   ├── InvestPage.tsx    # 投資授權主頁面
│   └── SuccessPage.tsx   # 成功確認頁面
└── index.ts              # 更新 tenantRegistry 註冊 test002
```

### 使用的共享資源

**Hooks:**
- `useWallet()` - 錢包連接檢測
- `useApprove()` - USDT 授權
- `useReferral()` - 推薦碼處理

**Components:**
- `LoadingSpinner` - 載入動畫
- `ErrorDisplay` - 錯誤訊息顯示
- `WalletButton` - 授權按鈕

**API:**
- `registerCustomer()` - 用戶註冊

### 業務流程

1. **頁面載入** → 檢測 TronWeb 錢包
2. **錢包檢測成功** → 顯示投資資訊和授權按鈕
3. **點擊授權按鈕** → 調用 approve() 授權 USDT
4. **授權成功** → 調用 registerCustomer() 註冊
5. **註冊成功** → 導向 /success 頁面
6. **任何步驟失敗** → 顯示錯誤訊息
